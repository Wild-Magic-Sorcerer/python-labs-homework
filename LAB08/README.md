### Задание 1

Напишите программу, которая запрашивает у пользователя строку и определяет, содержит ли она где-нибудь в себе номер телефона в формате "+x(xxx)x-xx-xx" (где x — цифра). Если номер найден — напечатайте его выделив код страны, код города и номер абонента. Если не найден — напечатайте сообщение "Номер не найден".

### Задание 2

Создайте скрипт, который получает на вход строку с датой в формате "ДД.ММ.ГГГГ" и заменяет все такие даты на универсальный вид (на просто буквы) "DD.MM.YYYY". Выведите результат на экран.

### Задание 3

Реализуйте программу, которая разбивает полученную от пользователя строку на слова и разделяет по всем знакам препинания (точка, запятая, вопросительный и восклицательный знаки, тире). Выведите список слов без пустых элементов.
