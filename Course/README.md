### Задания для курсовой работы

Разработайте полностью самостоятельный Python-пакет для решения задач, указанных в задании. Каждый пакет — самостоятельный полноценный проект с логичной и понятной структурой из трёх модулей, классами и их связями.

Пакет должен быть структурирован как настоящий проект: содержать несколько модулей, внутри которых реализованы как функции, так и классы с инкапсуляцией и наследованием. В модулях должна быть прописана документация. 

Пакет должен успешно устанавливаться при помощи pip (в виртуальное окружение). 

Пакет должен быть опубликован в публичный репозиторий GitHub. 

У него должно быть полное и краткое описание и версия (тег). 

Пакет должен содержать программу, которую после установки можно будет запустить из командной строки.

### Задания

**Вариант №1**. **Библиотека книг**

**Модули:**
- Модуль book.py: класс Book (название, автор, год, методы вывода, сравнения).
- Модуль library.py: класс Library (список книг), методы добавления, удаления, поиска по названию и автору, сортировки.
- Модуль user.py: класс User (читаемый список книг), методы взятия и возврата книг.

**Задачи:**
1. Добавить/удалить книгу, 
2. Найти книгу по названию/автору, 
3. Посчитать общее число книг, 
4. Вывести список книг пользователя, 
5. Проверка доступности книги в библиотеке.

**Вариант №2**. **Учёт расходов и доходов**

**Модули:**
- Модуль transaction.py: класс Transaction (дата, сумма, описание).
- Модуль expense.py: класс Expense (наследник Transaction).
- Модуль income.py: класс Income (наследник Transaction).

**Задачи:**
1. Добавить операцию, 
2. Рассчитать баланс, 
3. Фильтр операций по дате, 
4. Сводка по категориям, 
5. Вывод детального отчёта.

**Вариант №3**. **Анализ погоды**

**Модули:**
- Модуль data_loader.py: загрузка данных из файла/API.
- Модуль weather_day.py: класс WeatherDay (температура, осадки, влажность).
- Модуль weather_stats.py: класс WeatherStats (максимум, минимум, среднее, тренды).

**Задачи:**
1. Загрузить и сохранить данные, 
2. Подсчитать статистику, 
3. Найти экстремальные значения, 
4. Выявить тренды, 
5. Сформировать отчёт.

**Задачи:**
1. Добавить задачу,

3. Удалить задачу,

5. Сортировать задачи.
