### Задание №1

Напишите скрипт, который принимает обязательный позиционный аргумент — имя пользователя и выводит приветствие с этим именем.

### Задание №2

Создайте программу, принимающую два позиционных аргумента — числа (x и y) и необязательный флаг математической операции (add, sub, mul или div). Если флаг операции не указан - программа должна проcто вывести 2 входных числа.

### Задание №3

Реализуйте скрипт, принимающий список строк через аргумент командной строки и флаг подсчёта этих строк. Если флаг указан, вывести количество введённых строк, иначе — сами строки построчно.

### Задание №4

Создайте утилиту, которая принимает имя входного файла (-\-input) и имя выходного файла (-\-output). Утилита копирует содержимое одного файла в другой и выводит сообщение об успешной операции.

### Задание №5

Напишите программу с аргументом для подробного режима работы. В обычном режиме программа должна считать факториал переданного числа, с флагом подробности — выводить подробный процесс вычисления.
